What this page does

This portal reads and writes Inventory_Control_Workbook.xlsx directly in SharePoint through Microsoft's cloud API — the same co-authoring service Excel Online uses. Any number of people can log movements at the same time, from anywhere, with no sync conflicts or duplicate copies. The workbook stays the system of record; entries appear in it within seconds.

Daily use

Search → tap the item → tap what happened (Used, Received, Shipped…) → qty → Save. Initials are remembered on this device. Use ＋ New request for non-stock needs.

Sign-in

Use your normal work Microsoft account. Access follows SharePoint permissions — if you can open the workbook, you can use the portal. IT setup (one time, by admin) is described in Portal_Cloud_Setup_Guide.docx.

If a save fails

Lost signal or an expired session queues the entry locally — see the Activity tab for a Retry button. Nothing is lost.

Notes

Data refreshes on every save, on ↻ Refresh, and when you return to the tab. On-hand, status, and reorder quantities come from the workbook's own formulas, so this page and Excel always agree.
